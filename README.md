# Electronic Store E-commerce app
 Electronics E-commerce Store is a comprehensive web application designed for users to browse and purchase electronics. The platform
 aims to provide a seamless and user-friendly shopping experience, focusing on the following features :
 User authentication: Users can create accounts, log in, reset passwords, and update their profiles.
 Product Display: On the homepage, users can browse categories, view new products, and explore the best-selling products.
 Shopping Cart: Allows users to add products to the cart and manage purchases.
 Favorites: Users can save their favorite products for future reference.
 Cash Payment: Supports cash on delivery for purchases.
 Responsive Design: The website is designed to work seamlessly on all screen sizes.
 ## Technologies Used :
 - Frontend :
 - React.js:For building an interactive and responsive user interface.
 - CSS3 & Bootstrap:Responsive Design
 - React Router:Navigation & Routing
 - Axios:HTTP Requests
 - Context API:State Management
 - Formik & Yup: Form Handling & Validation

 - Backend :
 - Node Js:Used for server-side JavaScript execution, providing fast and scalable handling of requests.
 - Express.js:A web framework for Node.js that simplifies handling routes and requests, enabling a smooth API development process
 - MongoDB:A NoSQL database used for managing and storing product, user,and order data, chosen for its flexibility and scalability
 - Cloudinary:A cloud-based service for managing and optimizing images,allowing easy upload, transformation, and delivery of high-quality images across the platform
